HTTP Methods Testing

GET /get
Test GET requests and view request information
POST /post
Test POST requests with form data or JSON
PUT /put
Test PUT requests with data
DELETE /delete
Test DELETE requests
PATCH /patch
Test PATCH requests with partial data
HEAD /head
Test HEAD requests (headers only)
OPTIONS /options
Test OPTIONS requests

Status Codes

GET POST PUT PATCH OPTIONS /status/<code>
Return response with specific HTTP status code (e.g., /status/404). Supports multiple HTTP methods.
GET /status/random
Return response with random status code. Only supports GET method.

Request Inspection

GET /headers
Return request headers information
GET /ip
Return client IP address
GET /user-agent
Return user agent and browser information
GET /cookies
Return cookies sent by client

Cookie Management

GET /cookies/add
Add random cookies to response for testing
GET /cookies/clear
Clear all cookies from client

Response Inspection

GET /json
Return sample JSON data
GET /xml
Return sample XML data
GET /html
Return sample HTML data

Utilities

GET /delay/<seconds>
Return delayed response (max 60 seconds)

System

GET /health
Health check endpoint
GET /api
API information and endpoint list
