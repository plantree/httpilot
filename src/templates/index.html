HTTP Methods Testing

GET /get
Test GET requests and view request information
GET:curl "http://localhost:5000/get?param1=value1¶m2=value2"
POST /post
Test POST requests with form data or JSON
POST:curl -X POST "http://localhost:5000/post" -H "Content-Type: application/json" -d '{"key":"value","number":42}'
PUT /put
Test PUT requests with data
PUT:curl -X PUT "http://localhost:5000/put" -H "Content-Type: application/json" -d '{"update":"data"}'
DELETE /delete
Test DELETE requests
DELETE:curl -X DELETE "http://localhost:5000/delete"
PATCH /patch
Test PATCH requests with partial data
PATCH:curl -X PATCH "http://localhost:5000/patch" -H "Content-Type: application/json" -d '{"field":"updated_value"}'
HEAD /head
Test HEAD requests (headers only)
HEAD:curl -I "http://localhost:5000/head"
OPTIONS /options
Test OPTIONS requests
OPTIONS:curl -X OPTIONS "http://localhost:5000/options"

Status Codes

GET POST PUT PATCH OPTIONS /status/<code>
Return response with specific HTTP status code (e.g., /status/404). Supports multiple HTTP methods.
GET:curl -L "http://localhost:5000/status/301"
POST:curl -X POST "http://localhost:5000/status/404"
PUT:curl -X PUT "http://localhost:5000/status/200"
PATCH:curl -X PATCH "http://localhost:5000/status/500"
OPTIONS:curl -X OPTIONS "http://localhost:5000/status/204"
GET /status/random
Return response with random status code. Only supports GET method.
GET:curl "http://localhost:5000/status/random"

Cache Testing

GET /cache
Test HTTP caching. Returns 304 (Not Modified) if If-Modified-Since or If-None-Match headers are present, otherwise returns data with Last-Modified and ETag headers.
GET:curl "http://localhost:5000/cache"
Conditional:curl -H "If-Modified-Since: Thu, 01 Jan 1970 00:00:00 GMT" "http://localhost:5000/cache"
ETag:curl -H "If-None-Match: some-etag" "http://localhost:5000/cache"
GET /cache/<seconds>
Set Cache-Control header for specified number of seconds (e.g., /cache/300 for 5 minutes).
GET:curl -i "http://localhost:5000/cache/300"
1 hour:curl -i "http://localhost:5000/cache/3600"
GET /etag/<etag>
Test ETag handling. Returns 304 if If-None-Match matches, 412 if If-Match doesn't match (and not wildcard), otherwise returns normal response with the specified ETag.
GET:curl "http://localhost:5000/etag/test123"
Match:curl -H "If-None-Match: test123" "http://localhost:5000/etag/test123"
Wildcard:curl -H "If-None-Match: *" "http://localhost:5000/etag/test123"
If-Match:curl -H "If-Match: other-etag" "http://localhost:5000/etag/test123"

Dynamic Data

GET /delay/<seconds>
Return delayed response with timing information. Maximum delay is 60 seconds. Response includes actual delay time and timestamps.
GET:curl "http://localhost:5000/delay/3"
10 sec:curl "http://localhost:5000/delay/10"
Timing:time curl "http://localhost:5000/delay/5"
Max:curl "http://localhost:5000/delay/60"
Error:curl "http://localhost:5000/delay/120"

Cookie Management

GET /cookies
Return cookies sent by client
GET:curl -b "session=abc123;user=test" "http://localhost:5000/cookies"
GET /cookies/add
Add random cookies to response for testing
GET:curl -c cookies.txt "http://localhost:5000/cookies/add"
GET /cookies/clear
Clear all cookies from client
GET:curl -b cookies.txt -c cookies.txt "http://localhost:5000/cookies/clear"
